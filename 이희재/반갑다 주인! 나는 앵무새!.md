
## 반갑다 주인! 나는 앵무새!

### 문제 설명

- 입력을 받은데로 출력하는 앵무새 프로그램입니다.<br>


### 자료 구조

- var<br>
    - 타입 : 문자열
    - 저장 데이터 : 받은 문자열 전부
### 풀이 과정

>-문제를 풀기 위한 풀이 과정을 작성합니다.<br>
>-풀이 과정을 단계별로 나누어 적거나, 의사 코드(pseudo code) 형태로 작성합니다.<Br>

```txt
1. 입력을 받아 var 변수에 저장

2. var를 앵무새가 말하듯 출력 예) 앵무새 : 블라블라

3. var에 저장된 값이 무슨 자료형인지 출력

```

### 코드 구현
사용 언어 : 파이썬<br>

 
```python
# 풀이과정 1
var = input()

# 풀이과정 2
print("앵무새 :", var)

# 풀이과정 3
print(type(var))
```

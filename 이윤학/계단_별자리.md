## (계단 별자리)

### 문제 설명

- 계단 모양의 별자리를 반복문을 통해 출력하기<br>


### 자료 구조

- star<br>
    - 타입 : 문자열
    - 저장 데이터 : '*'


### 풀이 과정
예시 :
```txt
1. star라는 이름의 변수를 만들고 '*'(별모양) 넣기

2. 사진속 별 모양을 보고 반복할 횟수 정해서 반복문 작성하기

3. 반복한 횟수 만큼 곱해서 별 출력하기

```

### 코드 구현
예시 : 
```python
# 풀이과정 1
star = '*'

# 풀이과정 2,3
for i in range(6):
    print(star*i)
```

## (따라쟁이 앵무새)
>-문제의 이름을 위 괄호 부분에 작성하세요.

### 문제 설명

- 이용해서 입력 받은 값을 그대로 출력하는 앵무새 프로그램<br>


### 자료 구조

- (var)<br>
    - 타입 : 문자열
    - 저장 데이터 : 입력한 문자열 


### 풀이 과정
예시 :
```txt
1. input을 이용해 입력받아서 변수 var에 저장

2. 입력받은 값을 저장한 변수 var를 앵무새가 말을 print()문으로 출력

3. 입력받은 값을 저장한 변수 var의 타입을 print(type(var))로 출력

```

### 코드 구현
예시 : 
```python
# 풀이과정 1
var = input()

# 풀이과정 2
print("앵무새 :", var)

# 풀이과정 3
print(type(var))
```
